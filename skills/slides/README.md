# Slides Skill

Chart.js, 디자인 토큰, 반응형 레이아웃, 카피라이팅 공식을 활용한 전략적 HTML 프레젠테이션 생성 스킬입니다.

## 개요

- **이름:** `ckm:slides`
- **버전:** 1.0.0
- **작성자:** AX추진팀

## 주요 기능

- Chart.js 기반 데이터 시각화
- 디자인 토큰 기반 일관된 스타일링
- 반응형 슬라이드 레이아웃
- 전략적 카피라이팅 공식 적용
