# Mirae Asset Design Skill

미래에셋 공식 디지털 가이드라인에 기반한 디자인 스킬입니다. 웹/모바일 UI, 배너, 이메일, 보고서 등 미래에셋 브랜드 아이덴티티에 맞는 디자인을 생성합니다.

## 개요

- **이름:** `mirae-asset-design`
- **버전:** 1.0.0
- **작성자:** AI혁신팀

## 브랜드 요소

- **공식 컬러:** Orange `#F58220`, Blue `#043B72`
- **공식 서체:** Spoqa Han Sans
- **로고 사용 규정** 포함
- **UI 컴포넌트 스펙** 포함
