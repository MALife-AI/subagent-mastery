# 미래에셋 디자인 스킬

미래에셋 공식 디지털 가이드라인에 기반한 디자인 시스템. 웹, 모바일, 이메일, 배너, 보고서 등 모든 디지털 산출물에 미래에셋 브랜드 아이덴티티를 일관되게 적용.

## When to Use

- 미래에셋 브랜드 컬러/서체를 적용한 UI 컴포넌트 제작
- 웹사이트 헤더/푸터 레이아웃 설계
- 웹 배너(가로형/세로형/정방형) 디자인
- 웹 메일/뉴스레터 템플릿 제작
- 보고서/발표자료 레이아웃
- SNS 프로필 이미지, 모바일 아이콘 디자인
- 그래프/차트 색상 적용
- 이메일 서명 포맷 적용
- 브랜드 컴플라이언스 검토
