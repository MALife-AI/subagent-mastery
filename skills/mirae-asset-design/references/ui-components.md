# 미래에셋 UI 컴포넌트 가이드

> 출처: 미래에셋 디지털 가이드라인 (브랜드전략실, 2020.04)

## 웹 사이트 헤더 (Web Site Header)

- 웹 페이지 메인 상단에 고정되며 메인 메뉴를 노출
- 기본 포맷을 유지하는 것을 원칙
- 시각적 주목도를 높이기 위해 **Type A 활용 권장**
- 스크롤 다운 시 헤더 영역 색상을 반전

### Type A (오렌지 배경 — 권장)
```
┌─────────────────────────────────────────────────┐
│ [MIRAE ASSET 로고]    미래에셋  글로벌  사회공헌  PR  ≡ │
│  배경: #F58220 / 텍스트: #FFFFFF                    │
└─────────────────────────────────────────────────┘
```

### Type B (화이트 배경)
```
┌─────────────────────────────────────────────────┐
│ [MIRAE ASSET 로고]    미래에셋  글로벌  사회공헌  PR  ≡ │
│  배경: #FFFFFF / 텍스트: #043B72                    │
└─────────────────────────────────────────────────┘
```

### 서브 메뉴형 헤더
상단에 유틸리티 메뉴(HTS 다운로드, 공인인증/OTP, 로그인, 회원가입) 추가.

## 웹 사이트 푸터 (Web Site Footer)

- 회사 계열사 링크, 저작권, 연락정보 표시

## 파비콘 (Favicon)

- 배경을 투명처리하고 미래에셋 오렌지 컬러 메인의 포맷
- 사이즈: 16px x 16px 기본, 상하 좌우 여백 확보
- M 심볼 사용 (오렌지 + 블루 조합)

## 웹 배너 포맷 (Web Banner Format)

미래에셋 오렌지 컬러 메인의 포맷을 정확하고 일관되게 사용.

### 가로형 배너
```
┌──────────────────────────────────────┐
│ [카피 텍스트]          [MIRAE ASSET 로고]│
│                      배경: #F58220    │
│ [서브 카피]                            │
│                      배경: #043B72    │
└──────────────────────────────────────┘
```

### 세로형 배너
- 상단: 오렌지 영역 + 로고
- 중단: 카피 + 설명
- 하단: CTA 버튼 (오렌지 아웃라인)

### 정방형 배너
- 팝업/모달 형태
- Close + Go 버튼 조합
- 오렌지 배경 상단 + 화이트 본문

## 웹 메일 포맷 (Web Mail Format)

- 브랜드 이미지를 대외적으로 전달하는 중요한 매체
- 기본 포맷을 유지하는 것을 원칙

### 구조
```
┌──────────────────────────────────┐
│ MIRAE ASSET [로고]    MIRAE ASSET│  ← 오렌지 헤더
│ Market Letter                    │
├──────────────────────────────────┤
│ Research                         │
│ Global Market (오렌지 텍스트)       │
│                                  │
│ [본문 내용]                        │
│                                  │
├──────────────────────────────────┤
│ Copyright                        │
│ MIRAE ASSET [로고]               │  ← 블루 푸터
└──────────────────────────────────┘
```

## SNS 프로필 이미지

- Type A: 오렌지 배경 + 화이트 로고 (권장)
- Type B: 화이트 배경 + 컬러 로고
- 계정명(프로필명)은 계열사명을 명확하게 표시

## 모바일 아이콘

- Type A: 오렌지 배경 + 화이트 로고 (권장)
- Type B: 화이트 배경 + 컬러 로고
- 일관된 관리 필요

## 이메일 서명

- 보조서체인 '맑은고딕' 사용 (PC 환경 호환)
- 이메일 서명에 이미지 배너 사용 시 스팸메일로 분류될 수 있어 권장하지 않음
