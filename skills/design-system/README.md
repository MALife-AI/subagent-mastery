# Design System Skill

3계층 디자인 토큰 아키텍처(Primitive - Semantic - Component), 컴포넌트 스펙, CSS 변수 기반의 체계적 디자인 시스템 스킬입니다.

## 개요

- **이름:** `ckm:design-system`
- **버전:** 1.0.0
- **작성자:** AI혁신팀

## 주요 기능

- 3계층 토큰 아키텍처 (Primitive -> Semantic -> Component)
- CSS Variables 기반 토큰 관리
- 스페이싱/타이포그래피 스케일
- 컴포넌트 스펙 정의
- Tailwind CSS 통합
