## 새로 통합된 디자인 지식 (외부 소스)

UI-UX-Pro-Max 스킬에 다음 3가지 디자인 철학이 reference로 통합되었습니다:

- **design-engineering.md**: 디자인 엔지니어링 철학, 애니메이션 의사결정 프레임워크, 컴포넌트 빌딩 원칙

- **taste-skill-metrics.md**: 메트릭 기반 디자인, 3가지 조정 파라미터, RSC 우선 접근

이 지식들은 `/polish` 커맨드를 통해 통합 활용됩니다.
