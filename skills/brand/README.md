# Brand Skill

브랜드 보이스, 비주얼 아이덴티티, 메시징 프레임워크, 에셋 관리, 브랜드 일관성을 다루는 종합 브랜드 관리 스킬입니다.

## 개요

- **이름:** `ckm:brand`
- **버전:** 1.0.0
- **작성자:** AI혁신팀

## 활용 시점

브랜디드 콘텐츠 작성, 톤 앤 보이스 정의, 마케팅 에셋 생성, 브랜드 컴플라이언스 검토, 스타일 가이드 구축 시
