# UI Styling Skill

shadcn/ui(Radix UI + Tailwind), Tailwind CSS, Canvas 기반의 아름답고 접근성 높은 UI를 구축하는 스킬입니다.

## 개요

- **이름:** `ckm:ui-styling`
- **버전:** 1.0.0
- **작성자:** AX추진팀

## 활용 시점

- UI 빌드, 디자인 시스템 구현
- 반응형 레이아웃, 접근성 컴포넌트 (다이얼로그, 드롭다운, 폼, 테이블)
- 테마/컬러 커스터마이징, 다크 모드
- 비주얼 디자인/포스터 생성
- 일관된 스타일링 패턴 수립
