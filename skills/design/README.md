# Design Skill

로고, CIP(기업 아이덴티티 프로그램), 배너, 아이콘, 프레젠테이션, 소셜 포토 등을 생성하는 종합 디자인 스킬입니다.

## 개요

- **이름:** `ckm:design`
- **버전:** 2.1.0
- **작성자:** AX추진팀

## 기능

- 로고 생성 (55개 스타일, Gemini AI)
- CIP 프로그램 (50개 산출물, CIP 목업)
- HTML 프레젠테이션 (Chart.js)
- 배너 디자인 (22개 스타일, 소셜/광고/웹/프린트)
- 아이콘 디자인 (15개 스타일, SVG)
- 소셜 미디어 이미지 (HTML to screenshot, 멀티 플랫폼)

## 지원 플랫폼

Facebook, Twitter, LinkedIn, YouTube, Instagram, Pinterest, TikTok, Threads, Google Ads
