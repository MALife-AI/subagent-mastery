# /skill-maker

사용자의 스킬 아이디어를 받아 **인터뷰 → 설계 → 생성**의 3단계로 완성도 높은 스킬을 만들어주는 커맨드입니다.

## 실행

`skills/skill-maker/SKILL.md`의 전체 가이드를 따릅니다.

**핵심 흐름:**

1. **Phase 1 — 의도 포착**: 사용자 입력에서 핵심 의도 추출
2. **Phase 2 — 방향성 인터뷰**: 부족한 정보를 최대 5개 질문으로 보충 (필수)
3. **Phase 3 — 설계안 제시**: 파일 구조 + 목차를 보여주고 승인 요청 (필수)
4. **Phase 4 — 스킬 생성**: SKILL.md 및 부속 파일 작성
5. **Phase 5 — 완료 보고**: 사용 방법 안내

**사용자 입력**: $ARGUMENTS

`$ARGUMENTS`가 비어있으면 "어떤 스킬을 만들어볼까요? 한 문장이면 충분합니다!"로 시작합니다.

`$ARGUMENTS`가 있으면 Phase 1부터 바로 시작합니다.

**중요**: Phase 2(인터뷰)와 Phase 3(설계안)에서 반드시 사용자 확인을 받은 뒤 다음 단계로 넘어갑니다. 스킬을 바로 만들지 않습니다.
